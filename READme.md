## üí° Description
The objective of this project is developing a network application based on the
client/server architecture. You are required to build a web application that is used as a simple shopping
website. The website allows the users to lookup products descriptions, types, manufacturers‚Ä¶ etc. Users
should be allowed to create an account, add items to their ‚Äúcart‚Äù and search for products. 
<h3>Components</h3>
	<hr>
<ul>
	<li>Users Login (Main Page):Registered users should be allowed to log in to their accounts using their stored username and password. If an unregistered user tries to log in an error message should be displayed.</li>
	<li>User Registration: Users should be allowed to create an account using a unique username and a password and the users‚Äô information should be stored in a database using MongoDB. If the user tried to register using
an already taken username, an error message should be displayed.</li>
	<li>Home Page:The home page is the first page that should be encountered by the users when they log in to their accounts. It contains several item types and a button to view the user‚Äôs ‚Äúcart‚Äù. When the user clicks
on any item type, they should be redirected to that type‚Äôs page.</li>
	<li>Type Page:The type page contains all the items within this type. When a user clicks on any item‚Äôs name, they should be redirected to that item‚Äôs page.</li>
	<li>Item Page:
The item page contains a description for the item. The page should also contain an embedded link
for a video describing the item which can be streamed by the user. Please don‚Äôt copy the video
itself to the folder so that it doesn‚Äôt exceed the allowed size. Finally, an ‚Äúadd to cart‚Äù button should
be added. The button adds this item to the user‚Äôs ‚Äúcart‚Äù in the database.</li>
	<li>Cart Page:
The cart page contains the items that the user previously added using the ‚Äúadd to cart‚Äù button. A
‚Äúview cart‚Äù button should be added to the home page that directs the user to their own cart page.</li>
	<li>Search:
A search bar will be displayed in all pages except for the registration and login pages. The search
will be done using items names only. The search result is either an ‚Äúitem not found‚Äù message if the
item was not available in the database or a list of the items that contain the search keyword in their
names. The search results should be clickable and they direct you to that specific item‚Äôs page.</li>
</ul>
